
4. 首次进入可选择展示新手指引（可跳过）

### 1.3 购物车购物流程

```
┌─────────────────────────────────────────────────────────────┐
│                       购物流程                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  商城列表 → 商品详情                                         │
│     ↓                                                        │
│  选择规格 → 选择数量                                          │
│     ↓                                                        │
│  加入购物车 / 立即购买                                        │
│     ↓                                                        │
│  购物车页面 → 编辑数量 / 删除                                 │
│     ↓                                                        │
│  结算页面 → 确认地址 → 确认订单                               │
│     ↓                                                        │
│  支付页面 → 选择支付方式 → 支付                               │
│     ↓                                                        │
│  支付成功 → 跳转订单详情                                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**详细步骤说明：**

1. **商品列表页**
   - 点击分类标签筛选商品
   - 下拉刷新加载更多
   - 点击商品卡片进入详情

2. **商品详情页**
   - 左右滑动查看多张图片
   - 点击规格选择（弹出底部选择器）
   - 点击数量加减
   - 点击"加入购物车"添加到购物车
   - 点击"立即购买"直接跳转结算

3. **购物车页面**
   - 左滑商品显示删除按钮
   - 点击编辑数量
   - 全选/取消全选
   - 计算总价

4. **结算页面**
   - 选择收货地址（可新增）
   - 查看商品清单
   - 选择配送方式
   - 使用优惠券
   - 查看实付金额

5. **支付页面**
   - 选择支付方式（微信/支付宝/银行卡）
   - 确认支付
   - 支付成功/失败提示

6. **订单详情**
   - 查看订单状态
   - 查看物流信息（待发货/待收货）
   - 申请售后（待收货）
   - 确认收货

## 二、通用交互规范

### 2.1 手势操作

| 手势 | 应用场景 |
|------|----------|
| 点击 | 选择、确认、进入详情 |
| 长按 | 弹出操作菜单（删除、分享） |
| 左滑 | 列表项显示操作按钮 |
| 右滑 | 返回上一级 |
| 下拉 | 刷新列表 |
| 上拉 | 加载更多 |
| 双指缩放 | 图片预览缩放 |
| 左右滑动 | 轮播图切换、图片浏览 |

### 2.2 动画反馈

| 交互 | 动画效果 | 时长 |
|------|----------|------|
| 按钮点击 | scale(0.98) + 亮度变化 | 100ms |
| 卡片点击 | 阴影加深 | 150ms |
| 收藏点击 | 心形放大 + 粒子效果 | 300ms |
| 加入购物车 | 商品飞入购物车 | 400ms |
| 页面切换 | 淡入 + 上滑 | 250ms |
| 弹窗显示 | 缩放弹出 + 遮罩淡入 | 300ms |
| 加载更多 | 骨架屏 + 加载指示器 | - |
| 下拉刷新 | 珠粒下拉动画 | - |

### 2.3 页面跳转逻辑

```
┌─────────────────────────────────────────────────────────────┐
│                      页面跳转方式                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Tab切换：                                                  │
│  底部Tab点击 → 切换到对应模块（保留状态）                    │
│                                                             │
│  列表→详情：                                               │

```

## 四、错误处理

### 4.1 网络错误

- **无网络**：显示离线状态提示，提供刷新按钮
- **请求超时**：显示超时提示，提供重试按钮
- **服务器错误**：显示服务异常提示，建议稍后重试

### 4.2 数据错误

- **加载失败**：显示错误提示 + 重试按钮
- **解析错误**：显示数据异常提示
- **为空**：显示空状态插图 + 引导操作

### 4.3 操作错误

- **添加购物车失败**：显示错误Toast，提示原因
- **支付失败**：显示错误提示，提供重新支付入口
- **收藏失败**：显示错误Toast，自动重试

## 五、 Accessibility（无障碍）

### 5.1 触摸区域

- 最小点击区域：44x44px
- 按钮内边距：确保足够触摸区域
- 列表项高度：最小48px

### 5.2 视觉辅助

- 文字对比度：≥ 4.5:1
- 状态指示：颜色+图标双重指示
- 加载状态：文字+动画双重提示

### 5.3 语音支持

- 关键操作有语音反馈
- 图片有alt描述
- 支持屏幕阅读器
