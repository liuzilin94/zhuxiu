# 「数绘珠光」APP前端界面设计方案

> 泉州珠绣非遗文创数字化设计与销售平台  
> 版本：v1.0 | 日期：2026-03-17

---

## 一、项目概述

### 1.1 项目背景

「数绘珠光」是泉州珠绣非遗文创数字化设计与销售平台的核心APP产品，旨在通过数字化手段传播珠绣非遗文化，同时满足文创产品销售和用户参与设计的多元需求。项目秉承"非遗创新传承"的核心理念，将传统珠绣工艺与现代数字技术融合，打造兼具文化性、实用性和年轻化的移动端产品。

### 1.2 项目定位

- **核心定位**：珠绣非遗文化传播 + 文创产品销售 + 数字化设计体验
- **目标用户**：非遗文化爱好者、文创产品消费者、珠绣手工艺学习者
- **设计理念**：新中式国风与简约现代融合，兼顾文化内涵与年轻审美
- **项目名称**：数绘珠光
- **平台**：iOS/Android 移动端APP

### 1.3 核心功能模块

| 模块 | 功能描述 |
|------|----------|
| 文化传播模块 | 展示珠绣历史渊源、工艺技法、传承人故事，支持图文、视频形式 |
| 产品展示与销售模块 | 文创材料包、实用文创单品，支持购物车、支付、订单管理 |
| 数字化设计体验模块 | 珠绣纹样库、简单编辑功能、设计教程视频 |
| 互动交流模块 | 话题展示、用户作品分享、点赞评论互动 |
| 个人中心模块 | 会员积分、订单管理、收藏、个人资料设置 |

## 五、交互设计规范

### 5.1 通用交互规范

#### 点击反馈

- **按钮点击**：scale(0.98) 缩放 + 50ms过渡
- **卡片点击**：轻微阴影变化 + 100ms过渡
- **图标点击**：颜色变化 + 透明度变化

#### 珠光动效

- **加载动画**：珠粒渐变浮现
- **按钮悬停**：珠光扫过效果（从左到右，300ms）
- **页面切换**：淡入淡出 + 轻微上滑（250ms）

#### 手势操作

- **列表滑动**：左滑显示操作按钮（删除、收藏）
- **图片预览**：双指缩放 + 左右滑动切换
- **下拉刷新**：珠粒下拉指示器
- **上拉加载**：加载更多动画

### 5.2 核心功能交互流程

#### 购物车流程

```
商品列表 → 加入购物车 → 购物车页面 → 结算 → 支付 → 订单详情
```

#### 搜索流程

```
点击搜索框 → 输入关键词 → 实时联想 → 选择/搜索 → 结果列表 → 详情页
```

#### 收藏流程

```
点击收藏按钮 → 动画反馈（心形放大） → 加入我的收藏 → 可在个人中心查看
```

### 5.3 动画时序

```css
/* 页面切换 */
--transition-page: 250ms ease-out;

/* 按钮反馈 */
--transition-button: 100ms ease-out;

/* 卡片悬停 */
--transition-hover: 150ms ease-out;

/* 弹窗显示 */
--transition-modal: 300ms cubic-bezier(0.4, 0, 0.2, 1);

/* 加载动画 */
--transition-loading: 1500ms ease-in-out;
```

## 六、技术实现建议

### 6.1 响应式设计

- **适配屏幕**：iPhone SE - iPhone 15 Pro Max
- **安全区域**：适配刘海屏和圆角屏幕
- **最小宽度**：320px
- **最大宽度**：428px（平板等大屏设备）

### 6.2 性能优化

- **图片格式**：WebP优先，支持渐进式加载
- **懒加载**：图片延迟加载，首屏优先
- **骨架屏**：数据加载前显示占位
- **缓存策略**：静态资源缓存1年

### 6.3 无障碍设计

- **对比度**：文字与背景对比度 ≥ 4.5:1
- **触摸区域**：最小 44x44px
- **语音支持**：关键交互支持语音反馈
- **字体缩放**：支持系统字体大小调整
